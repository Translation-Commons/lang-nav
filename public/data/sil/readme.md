The Summer Language Institute (SIL) provides several datasets that are used in this project. These datasets are provided by SIL International and include information about languages, their vitality, and digital support.

**Citation**
Eberhard, David M., Gary F. Simons, and Charles D. Fennig (eds.). 2025. Ethnologue: Languages of the World. Twenty-eighth edition. Dallas, Texas: SIL International. Online version: https://www.ethnologue.com/credits/


## Ethnologue 2012 Data

This data only includes language vitality estimates based on the EGIDS (Expanded Graded Intergenerational Disruption Scale) scale as of 2012. See https://www.ethnologue.com/methodology/#Status for more information about the EGIDS scale.

The file [ethnologue2012.tsv](/public/data/sil/ethnologue2012.tsv) contains data that was publicly shared on the Ethnologue website in 2012. It was acquired and used under the terms of the Ethnologue license at that time by the Digital Language Death project (Andr√°s Kornai), used in their 2013 publication https://journals.plos.org/plosone/article?id=10.1371/journal.pone.0077056.

At the time, Ethnologue had 7,776 languages, but the data file contains 8,426 rows because some entries come from other datasets and were given manual vitality estimates by the Digital Language Death project.


The data was accessed 2012-02-28

## Ethnologue 2025 Data

The file [ethnologue2025.tsv](/public/data/sil/ethnologue2025.tsv) contains public information available from Ethnologue dataset, licensed from SIL International. This dataset includes language codes, language names, coarse population bands (eg. "10K to 1M"), coarse vitality status (4 levels), and digital support.

The data was accessed 2025-03-27.
